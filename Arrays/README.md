## 🔧 Реализованные функции

**Для лабораторной №14:**
### 1. `double* full_elements(double* array_14, int size)`
- **Назначение**: Заполнение массива значениями через функцию func(x)
- **Параметры**: 
  - `array_14` - указатель на массив
  - `size` - размер массива
- **Возвращает**: указатель на заполненный массив
  
### 2. `void print_array(double* array_14, int size)`
- **Назначение**: Вывод элементов массива на экран
- **Параметры**: 
  - `array_14` - указатель на массив
  - `size` - количество элементов для вывода
  
### 3. `double sum_array(double* array_14, int size)`
- **Назначение**: Вычисление суммы всех элементов массива
- **Параметры**: 
  - `array_14` - указатель на массив
  - `size` - количество элементов
- **Возвращает**: сумму элементов типа double
  
### 4. `double find_max_not_A(double* array_14, int size, double A)`
- **Назначение**: Поиск максимального значения, не равного А
- **Параметры**: 
  - `array_14` - указатель на массив
  - `size` - размер массива
  - `A` - значение для исключения их поиска
- **Возвращает**: максимальное значение, не равное А

**Для лабораторной №16:**
### 1. `double* create_random_array(int size)`
- **Назначение**: Создание динамического массива со случайными числами
- **Параметры**: 
  - `size` - размер создаваемого массива
- **Возвращает**: указатель на динамически выделенный массив
  
### 2. `double find_min_among_three(double a, double b, double c, int* valid_count)`
- **Назначение**: Нахождение минимального значения среди трех чисел
- **Параметры**: 
  - `a, b, c` - сравниваемые значения
  - `valid_count` - указатель на счетчик валидных значений
- **Возвращает**: минимальное значение среди трех
  
### 3. `double* create_d_array_min(double* a, int size_a, double* b, int size_b, double* c, int size_c, int* result_size)`
- **Назначение**: Создание массива по правилу d[i] = min(a[i], b[i], c[i]).
- **Параметры**: 
  - `a, b, c` - указатели на исходные массивы
  - `size_a, size_b, size_c` - размеры исходных массивов
  - `result_size` - указатель для возврата размера результатирующего массива
- **Возвращает**: указатель на новый массив d
  
### 4. `void print_array_info(double* array, int size, const char* name)`
- **Назначение**: Вывод информации о массиве с указанием его имени
- **Параметры**: 
  - `array` - указатель на массив
  - `size` - размер массива
  - `name` - имя массива для отображения
